#1 InfiAgent：长周期研究任务的文件状态引擎 9.4

Agent workflowResearch workflowSystem workflowAI workflow

价值判断：InfiAgent 的机会不是又一个 Agent 框架，而是把长周期任务的上下文从聊天窗口迁移到文件化 workspace state，适合解决日报、资料研究、项目拆解这类多轮任务越跑越乱的问题。

解决的问题：每日雷达容易丢状态、重复推荐、忘记昨天规则；文件状态引擎可以让每次运行先读历史快照，再输出增量结果。

主流方案对比：prompt chaining 适合短流程，RAG 适合检索资料；文件化状态更适合可恢复的长周期研究流水线。
